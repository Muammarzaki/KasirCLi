# KasirCLi

Sistem kasir berbasis CLi

<br>

## Fitur

- [x] penyimpanan local
- [x] computasi pentotalan barang yang di beli
- [ ] I/O berbasis cli dan mungkin akan di simpan di file.txt

<br>

## How to use

- Call Command
  ```bash
  ~$ ksr
  ```
- untuk memanggil option 
  ```bash
  ~$ ksr start
  ```
- untuk memanggil insert data from csv 
  ```bash
  ~$ ksr insert  
  ```
